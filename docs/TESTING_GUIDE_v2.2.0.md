## 10. 调试技巧（如果出问题）

### F12 Console 看错误
- 红色 `Error` 必报
- 黄色 `Warning` 看是否与 `zentype` 相关

### F12 Elements 检查 DOM
```
Ctrl+F 搜 "zentype-cursor"
```
- 应找到 `<div id="zentype-cursor" class="hidden ...">`
- 如果 `class` 包含 `hidden` 但光标可见 → CSS 问题
- 如果元素根本不存在 → 模块未初始化

### F12 Application → Storage → Local Storage
- 看是否有 `siyuan-zen` 相关条目
- 正常应该为空（v2 不存任何数据）

### 强制重新初始化
```
思源 → 设置 → 插件 → 焦点写作 → 禁用 → 启用
```
如果还不行：直接**重启思源**。

### 看 EventBus 是否订阅成功
F12 Console 输入：
```js
window.siyuan.eventBus
```
应该返回一个对象（具体结构看思源版本）。
